## Test Plan détaillé

### Test 1 : Dictée vocale
```
1. Ouvrir app mobile
2. Cliquer micro
3. Dicter "Appeler Fred pour la séance"
4. Vérifier que la tâche s'ajoute
5. Vérifier dans portail admin
```

### Test 2 : Filtres Pro/Perso
```
1. Ajouter 2 tâches : 1 Pro, 1 Perso
2. Cliquer "Pro" sur mobile
3. Vérifier que seule la Pro apparaît
4. Cliquer "Perso"
5. Vérifier que seule la Perso apparaît
6. Cliquer "Tous"
7. Vérifier que les 2 apparaissent
8. Répéter sur portail admin
```

### Test 3 : Real-time sync
```
1. Ouvrir app mobile d'un côté
2. Ouvrir portail admin d'un autre côté (ou autre navigateur)
3. Ajouter tâche sur mobile
4. Vérifier qu'elle apparaît dans portail < 1s
5. Cocher la tâche dans portail
6. Vérifier qu'elle est cochée sur mobile < 1s
```

### Test 4 : Offline
```
1. Ouvrir app mobile
2. Désactiver WiFi/4G
3. Ajouter une tâche (offline)
4. Vérifier qu'elle s'ajoute localement
5. Réactiver connexion
6. Vérifier que la tâche synchro à Supabase
```

### Test 5 : CEO CR
```
1. Ajouter 5 tâches (2 complétées, 3 en cours)
2. Exécuter agent CEO
3. Vérifier que le CR liste bien :
   - Tâches complétées
   - Tâches en cours
   - Tâches urgentes
4. Format lisible et correct
```
