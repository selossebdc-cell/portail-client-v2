📊 Synthèse exécutive

Taïna gère 3 parcours client parallèles :

🏖️ BtoC DIP (Contact@, réseaux sociaux) — Flux court, peu structuré, délai 4-5j
🌐 BtoC Info (Info@, site web) — Flux plus long (appels), moins de volume
🚢 B2B Ponant/UCPA (ttharsis@) — Flux structuré, pré-réservation 15j avant

🔴 Problème central : 0% automatisation, 3 boîtes mails non centralisées, facturation manuelle, 50 prestataires en WhatsApp, pas de cloud.

✅ Solution : Centraliser via Ezus (hub client + facturation) + Claude automatisé (réponses + qualification) + intégration WhatsApp pour prestataires.

🔄 Parcours 1 : BtoC DIP (Meurtre au Paradis)

14 étapes du lead au feedback post-excursion. Chaque étape peut être accélérée.

Détail des 14 étapes

1. LEAD ENTRANT
Via Facebook, Instagram, WhatsApp, SMS, email
Aujourd'hui : Taïna lit manuellement chaque canal
→ Idéal : 1 inbox centralisée (email + chat)
2. QUALIFICATION
Extraire : nom, date, nbre pax, budget, type tour
Aujourd'hui : Taïna détermine le type à la main
→ Idéal : Claude automatisé extrait les infos (gain 3-5 min)
3. RÉPONSE TYPE
Envoi catalogue ou infos
Aujourd'hui : Copier-coller depuis fichier Excel
→ Idéal : Claude génère brouillon en 30s (après 15j → envoi auto)
4. VÉRIFICATION DISPONIBILITÉ
Appels/WhatsApp aux 5 prestataires
Aujourd'hui : 45 min par excursion
→ Idéal : Chatbot WhatsApp → réponses en 2h (gain 40 min)
5. DEVIS / CONFIRMATION PRIX
Compiler coûts + générer devis
Aujourd'hui : Taïna compile manuellement
→ Idéal : Formule tarifaire dans Ezus → devis auto (gain 5-10 min)
6. PAIEMENT
50% ou 100% via Crédit Mutuel
Aujourd'hui : Lien bancaire + copier confirmation dans Excel
→ Idéal : Webhook CM → Ezus OU export CSV (gain 2-3 min)
7. FACTURE INDIVIDUELLE
Générer facture client
Aujourd'hui : 0 facture (perte comptable)
→ Idéal : Ezus auto-génère (gain 10 min + traçabilité)
8. RELANCE 50% RESTANT
Email 1 mois avant l'excursion
Aujourd'hui : Taïna oublie souvent
→ Idéal : Automation Ezus (gain 5 min + 100% des relances)
9. CHOIX DE MENU
Formulaire + confirmation prestataires
Aujourd'hui : Mail manuel, compilation manuelle
→ Idéal : Formulaire Ezus + Claude extrait (gain 5-10 min)
10. BON DE COMMANDE PRESTATAIRES
Créer + distribuer aux 5 prestataires
Aujourd'hui : 20-30 min par excursion (Word/Excel)
→ Idéal : Ezus auto-génère → WhatsApp à chaque prestataire (gain 20 min)
11. RAPPEL VEILLE (J-1)
Email + SMS à tous les prestataires + client
Aujourd'hui : Pas systématique
→ Idéal : Automation Ezus/WhatsApp (gain 15 min)
12. JOUR J (COORDINATION)
Gérer les impromptus
Aujourd'hui : Taïna gère tout (guide absent, port bloqué, etc.) = 45 min
→ Idéal : Chatbot WhatsApp pour questions standard (gain 30 min)
13. POST-EXCURSION (FEEDBACK)
Formulaire + photos + envoi
Aujourd'hui : Assistante fait parfois (non systématique)
→ Idéal : Formulaire Ezus + Claude recap auto (gain 5-10 min)
14. FIDÉLISATION
Email 6 mois après + nouvelles offres
Aujourd'hui : 0 (sauf si client re-contacte)
→ Idéal : Automation Ezus + newsletter (gain récurrent)
