📧 Configuration Apple Mail

Guide complet pour configurer tes 3 boîtes mail + archivage

✅ Avant de commencer

Important : Cette opération ne supprime rien. On va juste ajouter tes 3 boîtes mail dans Apple Mail en gardant Roundcube fonctionnel (tu peux revenir en arrière si besoin).

Ce qu'on va configurer

  • contact@guadeloupe-explor.com — Leads Meurtre au Paradis (BtoC)
  • info@guadeloupe-explor.com — Leads site web + B2B agences
  • ttharsis@guadeloupe-explor.com — B2B Ponant/UCPA + correspondance pro
⏱️ Durée synchro : La première synchro peut prendre 5-10 min (selon le volume). Ensuite, ça se fait en temps réel.

1️⃣ Ouvrir Apple Mail

Lancer Apple Mail

  1. Clique sur le Finder (coin bas-gauche du dock)
  2. Cherche "Mail" ou "Courrier" dans Applications
  3. Double-clique pour ouvrir

2️⃣ Ajouter la 1ère boîte mail (contact@)

Configuration contact@guadeloupe-explor.com

  1. En haut à gauche, clique sur Mail → Préférences (ou Mail → Settings si tu es en anglais)
  2. Clique sur l'onglet Comptes (Accounts)
  3. En bas à gauche, clique sur le + pour ajouter un compte
  4. Sélectionne Autre compte Mail (Other Mail Account)
  5. Remplis :

Infos à remplir

Nom complet : Taina THARSIS
Adresse email : contact@guadeloupe-explor.com
Mot de passe : [ton mot de passe Roundcube/hosting]
💡 Si tu ne te souviens pas du mot de passe, contacte renaux@flbserver.com (ton hébergeur)

Continuer la configuration

  1. Clique Suivant (Next)
  2. Apple va tester la connexion (⏳ attends 10-20 sec)
  3. Si ça demande IMAP vs POP → sélectionne IMAP
  4. Serveur entrant (IMAP) : guadeloupe-explor.com
  5. Port IMAP : 993 ou 143 (essaye 993 d'abord)
  6. Serveur sortant (SMTP) : guadeloupe-explor.com
  7. Port SMTP : 587
  8. Clique Ajouter un compte (Add Account)
✅ Premier compte ajouté ! Apple Mail va commencer à télécharger tes emails (peut prendre quelques minutes).

5️⃣ Organiser les boîtes dans Apple Mail

Créer des dossiers pour chaque boîte

  1. Dans la barre latérale gauche, tu vois maintenant 3 comptes (contact@, info@, ttharsis@)
  2. Développe chacun (clique sur la flèche pour voir les sous-dossiers)
  3. Pour chaque boîte, crée des dossiers :

Structure suggérée (par boîte)

  • 📧 Boîte de réception (Inbox)
  • 📁 En attente de paiement (emails avec lien de paiement non confirmé)
  • 📁 À confirmer (prestataires à vérifier)
  • 📁 Archivé (vieux emails, mais gardés pour référence)
💡 Pour créer un dossier : clique droit sur le compte → "Nouveau dossier" (New Folder)

6️⃣ Archivage Roundcube → Apple Mail

Récupérer tes vieux emails de Roundcube

  1. Va dans Roundcube (ton interface mail actuelle)
  2. Sélectionne les emails anciens que tu veux garder (ou utilise "Tous les emails")
  3. Clique droit → Exporter ou cherche un option "Télécharger"
  4. Un fichier .zip ou .mbox va se télécharger

Importer dans Apple Mail

  1. Dans Apple Mail, clique sur Fichier → Importer les boîtes aux lettres (Import Mailboxes)
  2. Sélectionne le fichier téléchargé depuis Roundcube
  3. Apple Mail va créer un dossier "Importer" avec tes vieux emails
  4. Tu peux ensuite les organiser dans les dossiers ci-dessus
⚠️ Si tu ne trouves pas l'option "Importer", envoie un message à renaux@flbserver.com pour qu'il t'explique comment exporter depuis Roundcube.

7️⃣ Vérifier la synchro

Contrôler que tout fonctionne

  1. Va dans chaque boîte de réception (contact@, info@, ttharsis@)
  2. Clique sur la 1ère boîte → tu dois voir tes emails
  3. Vérifie que les dates, les expéditeurs sont corrects
  4. Envoie-toi un test email → il doit arriver dans Apple Mail en quelques secondes
✅ Si tu vois tes emails dans Apple Mail, c'est que c'est bon !

8️⃣ Paramètres avancés (optionnel)

Personaliser Apple Mail

  • Signature email : Mail → Préférences → Signatures. Ajoute ta signature pro pour chaque boîte
  • Archiver au lieu de supprimer : Mail → Préférences → Compte, coche "Archive" au lieu de "Trash"
  • VIP : Ajoute tes clients importants (Ponant, UCPA, etc.) en VIP pour les voir en priorité

🎉 C'est fini !
