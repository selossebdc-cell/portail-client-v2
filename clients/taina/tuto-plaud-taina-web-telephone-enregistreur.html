🎙️ Tuto PLAUD pour Taïna

Reunions physiques, visios avec l'app PLAUD sans outil, et excursions (en attendant l'enregistreur)

Objectif du tuto

L'objectif est de capter tes reunions et debriefs sans perte d'information, meme avant reception de l'enregistreur physique.

  • Cas 1 : enregistrer une reunion physique.
  • Cas 2 : enregistrer une visio avec l'app PLAUD, sans enregistreur.
  • Cas 3 : enregistrer les excursions (brief, debrief, incidents, idees).
Le plus important : tu peux commencer tout de suite avec le telephone + app web.

1) Demarrage rapide (sans enregistreur physique)

A faire aujourd'hui

Etape A - Installer l'app sur le telephone

  1. Ouvre l'App Store (iPhone) ou Google Play (Android).
  2. Cherche PLAUD (ou PLAUD AI selon l'intitule affiche).
  3. Installe l'application, puis connecte-toi avec ton email habituel.

Etape B - Autoriser micro + notifications

  1. Accepte l'acces au micro quand l'app le demande.
  2. Active les notifications (utile pour savoir quand la transcription est prete).
Si le micro n'est pas autorise, l'enregistrement ne marchera pas correctement.

Etape C - Premier test (2 minutes)

  1. Lance un enregistrement test dans l'app.
  2. Parle 30-60 secondes (ex: planning de la semaine).
  3. Arrete l'enregistrement puis verifie que le texte est genere.

2) Cas 1 - Enregistrer une reunion physique (sans enregistreur, pour l'instant)

Mode provisoire recommande

Avant la reunion (2 min)

  1. Ouvre l'app PLAUD sur le telephone.
  2. Place le telephone pres du centre de table, ecran vers le bas, mode silencieux.
  3. Lance l'enregistrement 1 minute avant le debut.

Pendant la reunion

  1. Parle clairement, une personne a la fois si possible.
  2. Au debut, dicte une intro courte:
"Reunion X, date, participants, objectif."

Apres la reunion (3 min)

  1. Arrete l'enregistrement et verifie la synchronisation.
  2. Renomme la note au format ci-dessous.
AAAA-MM-JJ - Reunion physique - Sujet - Actions
Astuce: en attendant l'enregistreur, le telephone remplace 100% du materiel pour ce type de reunion.

3) Cas 2 - Enregistrer une visio avec l'app PLAUD (sans enregistreur)

Workflow temporaire simple

Option la plus simple (recommandee)

  1. Lance la visio sur l'ordinateur.
  2. Pose le telephone a cote des enceintes du Mac/PC.
  3. Lance l'enregistrement PLAUD sur telephone.
  4. Verifie les 10 premieres secondes (niveau sonore OK).

Debut de note conseille

"Visio avec l'app PLAUD, date, sujet, objectif de la session."

Fin de note conseille

"Resume final: decisions, prochaines actions, echeances."
Si la qualite est faible: monte legerement le volume de sortie ordinateur et rapproche le telephone.

4) Cas 3 - Enregistrer les excursions

A utiliser tous les jours terrain

Quand enregistrer ?

  • Avant excursion : briefing logistique (participants, timing, prestataires).
  • Pendant excursion : incidents, changements, informations importantes.
  • Apres excursion : debrief a chaud (ce qui a marche / a corriger).

Format de nom recommande

AAAA-MM-JJ - Excursion - Type - Lieu/Sujet
Ex: 2026-04-29 - Excursion - Debrief - Pointe-des-Chateaux

Structure vocale efficace (60 sec)

Contexte / Probleme / Action prise / Suite a faire

5) Utiliser l'app web PLAUD (classement et exploitation)

A faire 1 fois par jour

Etape A - Ouvrir l'app web

  1. Connecte-toi a l'app web PLAUD depuis ton navigateur.
  2. Verifie que les enregistrements du telephone sont bien synchronises.

Etape B - Renommer proprement chaque note

Format conseille pour tout retrouver vite :

AAAA-MM-JJ - Client/Projet - Type - Sujet
Ex: 2026-04-29 - Guadeloupe Explor - Appel - Suivi prestataires

Etape C - Tri minimal en 3 dossiers

  • Clients en cours
  • Operations internes
  • Idees et contenu

Etape D - Extraire les actions

  • Pour chaque note importante: recopie 3 elements max dans ton outil de pilotage:
  • Decision prise
  • Action suivante
  • Deadline

6) Quand l'enregistreur physique PLAUD arrive

Transition vers usage avance

Etape A - Appairage initial

  1. Charge completement l'enregistreur avant la premiere utilisation.
  2. Ouvre l'app PLAUD sur telephone.
  3. Ajoute le device via Bluetooth selon l'assistant de l'app.
  4. Fais un test de 1 minute puis synchronise.

Etape B - Quel mode choisir ?

  • Telephone seul : idees rapides, notes perso, debrief journee.
  • Enregistreur physique : reunions longues, deplacements, contexte bruyant.
  • App web : tri, renommage, exploitation des comptes-rendus.

Etape C - Routine cible (simple)

  1. Capturer (telephone ou enregistreur)
  2. Synchroniser
  3. Renommer
  4. Extraire 1 a 3 actions concretes
Avec cette routine, aucune info importante ne se perd, et chaque enregistrement produit une action utile.

7) Check-list rapide anti-galere

  • Le micro est-il autorise dans les reglages du telephone ?
  • Le Bluetooth est-il actif (pour l'enregistreur physique) ?
  • L'app est-elle bien connectee au bon compte ?
  • La note est-elle bien synchronisee avant fermeture ?
  • Le nom de fichier est-il clair (date + sujet) ?
Confidentialite: avant tout enregistrement d'appel ou reunion, pense a prevenir l'interlocuteur si necessaire selon le contexte.

Plan de demarrage en attendant l'enregistreur

  • Jour 1: installation + test reunion physique + test visio avec l'app PLAUD.
  • Jour 2: 1 enregistrement excursion + tri dans l'app web.
  • Puis chaque jour: 1 capture utile + 1 extraction d'actions.
  • A l'arrivee de l'enregistreur: appairage + test + bascule progressive.
