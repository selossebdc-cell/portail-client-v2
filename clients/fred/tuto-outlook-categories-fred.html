👆 Les 3 étapes — Fred, tu cliques
1
Créer les 5 catégories couleurs
⏱ ~10 min
📍 Le chemin dans Outlook :
Onglet Accueil → Catégoriser → Toutes les catégories → Nouveau
1️⃣ Dans la fenêtre qui s'ouvre → clic sur "Nouveau"
✏️ Tape le nom : "V8" → choisis la couleur Verte → OK
🔁 Répète pour chaque activité : 🔴 Fu-Fight → Rouge 🟠 Sourcing → Orange 🔵 Perso → Bleu foncé ⚙️ Infos → Gris
✅ Quand les 5 sont créées → clic "OK" pour fermer la fenêtre
💡 Si tu ne vois pas l'onglet "Accueil" → clique d'abord sur un mail dans ta boîte de réception. L'onglet apparaît automatiquement.
2
Appliquer les catégories sur tes mails
⏱ ~15 min
📍 Sur chaque mail à classer :
Clic droit sur le mail → Catégoriser → Choisir la couleur
🖱️ Clic droit sur un mail → "Catégoriser" → clique sur la bonne catégorie
🎯 Exemple : mail de ton fournisseur V8 → 🟢 V8. Mail de HSBC → 🔴 Fu-Fight
📦 Pour classer plusieurs mails d'un coup : tiens Ctrl enfoncé + clic sur chaque mail → puis clic droit → Catégoriser
👀 Tu vois une pastille couleur apparaître à gauche du mail ? C'est bon, c'est classé ✅
🚨 Pour Catherine : Fred fait les 2-3 premiers seul pendant que tu guides à voix. Ce qui reste = son action pour la semaine. Ne pas tout finir aujourd'hui.
3
Regrouper les mails par catégorie dans l'affichage
⏱ ~2 min
📍 Le chemin dans Outlook :
Onglet Affichage → groupe "Organisation" → Catégories
🖱️ Clique sur l'onglet "Affichage" en haut → cherche le bouton "Catégories" dans la barre → clique dessus
✨ Tes mails se regroupent automatiquement : tous les verts ensemble, tous les rouges ensemble, etc.
▶️ Tu peux réduire ou ouvrir chaque groupe en cliquant sur la petite flèche à gauche du nom du groupe
🏆 C'est le moment "waouh" à montrer à Fred. 1 clic et tout se range par activité. Prends 30 secondes pour qu'il voit le résultat avant de passer à la suite.
+
BONUS — Règles automatiques (2 max en séance)
⏱ ~10 min
📍 Le chemin dans Outlook :
Onglet Accueil → Règles → Gérer les règles et alertes → Nouvelle règle
1️⃣ Choisis "Appliquer la règle aux messages que je reçois" → Suivant
2️⃣ Condition : "De personnes ou groupe public" → clique sur le lien souligné → saisis l'adresse mail du fournisseur → OK
3️⃣ Action : "Affecter à la catégorie" → clique sur le lien souligné → choisis la bonne catégorie → OK
4️⃣ Clic "Terminer". La règle s'applique automatiquement à tous les prochains mails de cet expéditeur.
⚠️ 2 règles max en séance (ex: ton fournisseur V8 principal + HSBC). Le reste = Fred le fait seul dans la semaine en répétant le même chemin. Le mécanisme est compris, c'est l'essentiel.
