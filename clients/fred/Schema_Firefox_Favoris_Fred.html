🗑️ Ce qu'on supprime de tes favoris (et pourquoi)
📧
Les mails bookmarkés Un mail mis en favori = introuvable le lendemain. Si c'est important, mets le document dans OneDrive. Sinon, laisse-le dans ta boîte mail.
📄
Les fichiers locaux (C:\Users\fredf\...) Ces liens ne fonctionnent que sur TON PC. Sur ton iPad ou un autre ordi → erreur. Solution : mettre le fichier dans OneDrive.
🔗
Les liens PDF Google Viewer temporaires Ces URLs meurent après quelques heures. Inutile. Le vrai fichier doit être dans OneDrive.
💀
Les liens zombies (ASUS IE, ChatbotGPT tracking, doublons) Des vieux favoris importés depuis Internet Explorer, des liens pub. Ça encombre. On nettoie.
🎨
Même couleur que tes containers
V8 = vert, Fu-Fight = rouge, Sourcing = orange, Perso = bleu. C'est TOUJOURS les mêmes couleurs — dans Firefox, dans OneDrive, partout. Un seul système à retenir.
📌
Barre de favoris = raccourcis rapides
Dans chaque dossier, mets MAXIMUM 8-10 liens. Ce que tu ouvres plusieurs fois par semaine seulement. Le reste → dans OneDrive ou dans ta boîte mail.
🔄
Nettoyage 1x par mois (5 min)
Chaque début de mois : supprimer les favoris que tu n'as pas utilisés. Si ça fait 30 jours que tu ne l'as pas ouvert, c'est que t'en as pas besoin en favori.
