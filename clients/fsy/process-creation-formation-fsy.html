Idee & Validation

Avant de faire quoi que ce soit — valider l'idee
1-2 jours
▼
💡 Clarifier l'idee
1
Formuler l'idee en 1 phrase
Template : "J'aide [public cible] a [transformation] grace a [methode/format] en [duree]."
Exemple : "J'aide les femmes stressees a retrouver un sommeil profond grace a un programme de yoga en 4 semaines."
2
Identifier la marque porteuse
Est-ce que ca s'inscrit sous Face Soul Yoga (yoga, bien-etre, methode) ou Aurelia Del Sol (developpement perso, marque personnelle) ?
Cette decision impacte le branding, la page de vente et la communication.
✅ Checklist de validation
3
Demande du public
Verifier dans les DMs Instagram, commentaires YouTube, questions Telegram/WhatsApp.
Minimum 5-10 demandes ou questions similaires sur le sujet.
Optionnel : creer un sondage en story Instagram ou Telegram.
Instagram Telegram YouTube
4
Analyse concurrence
Lister 3-5 formations similaires. Comparer : prix, duree, format, positionnement.
Identifier ton angle unique (methode FSY, approche holistique, accompagnement personnalise).
5
Estimation du chiffre d'affaires
Calculer : prix x nombre inscriptions estime = CA previsionnel
Exemple : 399 EUR x 30 inscrits = 11 970 EUR
Couts a deduire : hebergement, outils, publicite, temps de creation.
📝 Notes Catherine
Ne pas sauter cette phase ! Beaucoup de creatrices passent des semaines a creer une formation que personne ne veut. La validation pre-creation est le meilleur investissement de temps.
Astuce : un simple sondage en story "Ca vous interesserait un programme sur [sujet] ?" avec un curseur emoji donne deja un signal.
⏱️ Estimation timeline
6
Planifier le retroplanning
Fixer la date de lancement souhaitee. Remonter en arriere pour planifier chaque phase.
Cycles recommandes : septembre (rentree) et mars (printemps/renouveau).
Prevoir 6-10 semaines minimum entre le GO et le lancement.
🚦
Gate GO / NO-GO
Tous les criteres doivent etre valides pour passer a la Phase 1
✓ Idee en 1 phrase ✓ Marque identifiee ✓ Demande validee (5+ signaux) ✓ Concurrence analysee ✓ CA previsionnel calcule ✓ Date de lancement fixee
1

Conception & Contenu

Structurer, filmer, monter — creer la formation
2-4 semaines
▼
🎯 Definir les objectifs pedagogiques
1
Learning outcomes (resultats concrets)
Ecrire 3-5 phrases commencant par "A la fin de cette formation, l'eleve sera capable de..."
Exemples : "...pratiquer une routine de 15 min pour dormir", "...identifier ses declencheurs de stress".
Regle : chaque outcome doit etre mesurable et concret.
📚 Structurer le programme
2
Decouper en modules / semaines
Chaque module = 1 theme cle = 1 transformation partielle.
Format recommande : 4-8 modules, 2-4 lecons par module, 10-20 min par lecon video.
Creer une arborescence : Module > Lecons > Support.
3
Scripter le contenu de chaque lecon
Pour chaque video : bullet points des points cles (pas un script mot-a-mot).
Inclure : intro (pourquoi ce sujet), contenu principal, exercice pratique, recap.
Prevoir les pratiques guidees et les meditations separement.
Canva
🎬 Production video
4
Setup et tournage
Materiel : camera/iPhone + trepied + ring light + micro-cravate/Rode.
Decor : fond coherent avec la marque (plantes, tissus, eclairage chaud).
Format : 16:9 horizontal pour les lecons, 9:16 vertical pour les teasers Instagram.
Tips : filmer par batch (toutes les videos d'un module en une seance).
5
Montage avec CapCut
Ajouter : intro animee (5s), sous-titres, transitions douces, musique de fond (libre de droits).
Exporter en 1080p minimum. Poids ideal : < 500 Mo par video.
CapCut
6
Upload sur Bunny.net
Uploader chaque video dans un dossier dedie a la formation.
Nommage : FSY-[formation]-M[module]-L[lecon].mp4
Exemple : FSY-sommeil-M01-L02.mp4
Recuperer les liens embed pour integration Kajabi.
Bunny.net → Stream → Video Library → Upload → Copier Embed Code
Bunny.net
📄 Supports complementaires
7
Creer les PDFs et workbooks dans Canva
Templates a creer : guide de bienvenue, fiche recap par module, workbook exercices, journal de suivi.
Charte graphique coherente avec la formation (couleurs, polices, logo).
Exporter en PDF, taille optimisee pour mobile.
Canva
📱 Controle qualite
8
Tester le visionnage sur mobile
Verifier : lisibilite des sous-titres, qualite audio, temps de chargement.
80%+ de tes eleves regarderont sur telephone — c'est la reference.
📝 Notes Catherine
Ne vise pas la perfection au 1er tour ! Mieux vaut une V1 "assez bien" lancee qu'une V2 parfaite jamais terminee. Tu pourras toujours re-filmer des lecons apres feedback des premiers eleves.
Batch filming : bloque 2-3 demi-journees pour filmer tout un module. Change juste de haut pour donner l'illusion de jours differents.
🚦
Gate Phase 1
Tout le contenu est pret avant de toucher a Kajabi
✓ Outcomes definis ✓ Modules structures ✓ Videos montees + uploadees ✓ PDFs crees ✓ Test mobile OK
2

Plateforme Kajabi

Mettre en place la formation sur Kajabi
2-3 jours
▼
🏗️ Creer le produit
1
Creer le produit (Course) dans Kajabi
Choisir le type : Online Course (drip) ou Coaching (acces immediat).
Renseigner : titre, description, image de couverture (1280x720).
Kajabi → Products → + New Product → Online Course
Kajabi
2
Ajouter les modules et lecons
Pour chaque module : creer une categorie. Pour chaque lecon : ajouter un post.
Integrer les videos Bunny.net via embed code (Custom Code block).
Attacher les PDFs en telechargement dans chaque lecon.
Product → + Add Category → + Add Post → Custom Code (embed Bunny)
⏰ Configuration drip content
3
Programmer le deblocage progressif
Si formation sur X semaines : configurer le drip (1 module debloque par semaine).
Cela empeche l'eleve de "binge-watch" et favorise l'integration.
Exception : l'Abonnement Studio reste en acces libre.
Product → Settings → Drip → Schedule release
💳 Offres et tarification
4
Creer l'offre (Offer) dans Kajabi
Configurer : prix, plans de paiement (1x, 3x, mensuel), early bird si applicable.
Ajouter un coupon de lancement si prevu.
Exemples tarifs existants : Master The Method = 1950 EUR (possible 2900 EUR), Sommeil = 399 EUR, Studio = 17 EUR/mois.
Kajabi → Sales → Offers → + New Offer
5
Configurer la page checkout
Personnaliser la page de paiement : logo, couleurs FSY, temoignages, garantie.
Activer Stripe/PayPal si pas deja fait. Verifier le flux de bout en bout.
✉️ Emails automatiques Kajabi
6
Configurer les emails natifs Kajabi
Email de bienvenue : acces, mot de passe, lien formation, message perso d'Aurelia.
Deblocage module : notification quand un nouveau module est disponible.
Completion : felicitations + prochain pas (temoignage, upgrade, communaute).
Note : les sequences de nurturing plus complexes seront dans Brevo (Phase 4).
🎓 Certification (optionnel)
7
Configurer le certificat de completion
Si applicable (Master The Method) : activer le certificat Kajabi.
Personnaliser le design avec le branding FSY.
Definir les criteres : toutes les lecons completees + quiz passes.
Product → Settings → Certificate → Enable & Customize
💡 Astuce
Avant de lancer, fais un test complet : cree un compte test, achete avec un coupon 100%, verifie le parcours eleve de A a Z (bienvenue, drip, videos, PDFs, certificat).
🚦
Gate Phase 2
La formation est 100% fonctionnelle sur Kajabi
✓ Produit cree ✓ Drip configure ✓ Offre + checkout OK ✓ Emails configures ✓ Test complet effectue
3

Page de Vente & Tunnel

Convertir les visiteurs en acheteurs
3-5 jours
▼
✍️ Copywriting de la page de vente
1
Rediger le contenu de la page
Structure eprouvee pour la page de vente :

1. Hook : phrase d'accroche qui parle au probleme ("Tu n'arrives plus a dormir ? Tu te reveilles epuisee ?")
2. Douleurs : lister 3-5 frustrations que la cible ressent au quotidien
3. Vision : decrire la transformation ("Imagine te reveiller naturellement, pleine d'energie...")
4. Programme : contenu module par module avec les benefices
5. Temoignages : 3-5 avis (video si possible, screenshot sinon)
6. A propos : qui est Aurelia, pourquoi cette methode
7. FAQ : 5-8 questions frequentes (paiement, acces, niveau, remboursement)
8. CTA : bouton d'achat clair, visible, repete 3x dans la page
🎨 Design de la page
2
Creer la page de vente sur Wix ou Kajabi
Wix : si la page est sur facesoulyoga.com (recommande pour le SEO et la coherence).
Kajabi : si tu preferes un funnel integre (checkout sur la meme plateforme).
Design mobile-first. Photos haute qualite d'Aurelia en action (yoga, coaching).
Couleurs et typo coherentes avec le branding FSY.
Wix Kajabi Canva
3
Creer le flux d'achat (checkout flow)
Page de vente → Bouton CTA → Page checkout Kajabi → Confirmation.
Si la page de vente est sur Wix, le bouton redirige vers le checkout Kajabi (lien direct de l'Offer).
💰 Paiement & Confirmation
4
Verifier le paiement Stripe/PayPal
S'assurer que Stripe est connecte a Kajabi et que les virements tombent sur le bon compte.
Tester avec une carte de test Stripe (mode test).
5
Page de confirmation / merci
Creer une page post-achat : felicitations, prochaines etapes, lien vers la formation, invitation au groupe Telegram, lien de telechargement du guide de bienvenue.
📱 Optimisation mobile
6
Tester sur mobile (iOS + Android)
Verifier : vitesse de chargement, lisibilite, taille des boutons CTA, images responsives.
Tester le parcours complet : landing → scroll → CTA → checkout → confirmation.
70%+ du trafic viendra d'Instagram = mobile.
📝 Notes Catherine
La page de vente est l'element le plus important du tunnel. Passe du temps sur le copywriting. Regle d'or : parle de la transformation, pas des fonctionnalites. "Tu vas retrouver un sommeil profond" > "4 modules de yoga".
Les temoignages video convertissent 2x plus que le texte seul.
🚦
Gate Phase 3
Le tunnel de vente est complet et teste
✓ Copy redigee ✓ Page designee ✓ Checkout fonctionnel ✓ Page confirmation OK ✓ Test mobile passe

Automatisations & Contrats

Email sequences, e-signatures, chatbot, ManyChat
2-3 jours
▼
✉️ Sequences email Brevo
1
Sequence de nurturing pre-lancement
4-6 emails sur 2 semaines avant le lancement :
J-14 : teaser ("Quelque chose arrive...")
J-10 : partage de valeur (article, video gratuite liee au sujet)
J-7 : annonce officielle + early bird
J-3 : temoignage / cas client
J-1 : dernier rappel early bird
Jour J : lancement !
Brevo
2
Sequence de lancement (semaine de vente)
5-7 emails pendant la semaine de lancement :
Email 1 : portes ouvertes + lien
Email 2 : FAQ + objections
Email 3 : temoignages
Email 4 : behind the scenes (pourquoi cette formation)
Email 5 : dernieres 48h
Email 6 : derniere chance (fermeture ce soir)
Segmenter : audience chaude (liste existante) vs froide (nouveaux leads).
3
Sequence post-achat (onboarding)
J+0 : bienvenue + acces formation + groupe Telegram
J+3 : "Comment ca se passe ? Besoin d'aide ?"
J+7 : rappel module 2 + tips de pratique
Mi-parcours : encouragement + temoignage
Fin de programme : felicitations + demande temoignage + offre upgrade
4
Sequence fin d'acces / relance churn
Si acces limite dans le temps :
J-30 avant fin : rappel de finaliser
J-7 : derniere chance + offre extension
J-1 : acces ferme demain
Pour les abandons en cours de route : email de re-engagement a J+14 d'inactivite.
⚠️ Point critique
Pense a configurer les listes et tags Brevo AVANT de creer les sequences. Tag minimum : achat-[nom-formation], lead-[nom-formation], prospect-chaud. Ca evitera de spammer les mauvaises personnes.
📝 Contrats & e-signatures
5
Automatiser la signature electronique
Objectif : a l'achat sur Kajabi, le contrat/CGV est envoye automatiquement pour signature.

Option A — DocuSign :
Kajabi webhook (achat) → Zapier/n8n → DocuSign (envoyer contrat pre-rempli).

Option B — eSignatures.io :
API plus simple et moins chere. Meme logique : webhook → API eSignatures → email contrat.

Option C — Kajabi natif :
Si la formation inclut des termes acceptes au checkout (Terms of Service checkbox).
Moins juridiquement solide mais plus simple pour les petits montants (< 500 EUR).
DocuSign Zapier / n8n Kajabi
📝 Notes Catherine
Pour les formations < 500 EUR (ex : Sommeil 399 EUR), une checkbox CGV au checkout suffit. Pour les formations premium (> 1000 EUR, ex : Master The Method), un contrat signe est recommande. On mettra en place l'automatisation e-signature ensemble en session.
🤖 ManyChat & Instagram
6
Configurer l'automatisation ManyChat
Creer un flow ManyChat declenche par mot-cle en DM Instagram (ex : "SOMMEIL").
Le flow : envoie un message avec le lien de la page de vente + capture l'email pour Brevo.
Ajouter en story : "Ecris SOMMEIL en DM pour recevoir le lien".
ManyChat → Automation → + New Flow → Trigger: Instagram Keyword
ManyChat Instagram
💬 Telegram & WhatsApp
7
Mettre a jour le chatbot Telegram
Ajouter les infos de la nouvelle formation dans la base de connaissances du chatbot.
Le chatbot doit pouvoir repondre : prix, contenu, lien d'inscription, FAQ.
Creer un message d'annonce pour le groupe Telegram communaute.
Telegram WhatsApp
⚙️ Automations Kajabi
8
Tags et pipelines Kajabi
A l'achat : tagger automatiquement le contact (achat-[formation]).
Retirer le tag prospect-[formation] si existant.
Declencher l'ajout dans le pipeline CRM si applicable.
Kajabi → Automations → + New Automation → When: Purchase → Then: Add Tag
🚦
Gate Phase 4
Toutes les automatisations sont testees
✓ Sequences Brevo creees ✓ E-signature automatisee ✓ ManyChat configure ✓ Chatbot Telegram MAJ ✓ Tags Kajabi OK ✓ Test end-to-end passe

Lancement

Pre-launch, early bird, semaine de vente, full price
1-2 semaines
▼
🔥 Pre-lancement (J-14 a J-1)
1
Teasing Instagram
Stories : countdown sticker, behind-the-scenes du tournage, sneak peeks.
Reels : 2-3 reels educatifs sur le sujet de la formation (apporter de la valeur).
Posts : carousel "5 signes que tu as besoin de [sujet]".
Frequence : 1 story/jour + 2-3 reels/semaine pendant 2 semaines.
Instagram Canva CapCut
2
Emails de pre-lancement via Brevo
Activer la sequence de nurturing pre-lancement (creee en Phase 4).
Segmenter : audience chaude (abonnes, anciens clients) vs liste froide (leads).
Brevo
3
Waitlist / early bird
Ouvrir une liste d'attente (formulaire Brevo ou page Wix).
Offrir un bonus early bird : reduction 10-20%, module bonus, session live offerte.
Limiter l'early bird (nombre de places ou duree) pour creer l'urgence.
🚀 Semaine de lancement (Jour J a J+7)
4
Jour J — Email blast + post annonce
Envoyer l'email de lancement a toute la liste (segmente).
Poster l'annonce sur Instagram (reel + post + stories).
Activer le flow ManyChat (mot-cle en DM).
Annonce dans le groupe Telegram + WhatsApp.
Brevo Instagram ManyChat Telegram
5
Contenu quotidien pendant la semaine
Jour 1 : annonce officielle
Jour 2 : FAQ + reponse aux objections
Jour 3 : temoignages / resultats
Jour 4 : coulisses / histoire personnelle d'Aurelia
Jour 5 : rappel early bird qui se termine
Jour 6 : live Q&A (Instagram ou YouTube)
Jour 7 : dernier jour / fermeture
6
Live Q&A / webinaire (optionnel mais recommande)
Organiser un live de 30-45 min : presenter la formation, repondre aux questions en direct.
Plateforme : Instagram Live, YouTube Live ou Zoom.
Inclure un CTA a la fin avec le lien d'inscription.
Tip : le live est souvent le declencheur de vente le plus puissant.
Instagram YouTube
💵 Passage au plein tarif
7
Desactiver l'early bird, activer le prix normal
Desactiver le coupon / offre early bird dans Kajabi.
Mettre a jour la page de vente (retirer les mentions early bird).
La formation reste en vente en mode "evergreen" ou fermer les inscriptions.
🎉 Conseil de lancement
Les 3 moments qui generent le plus de ventes : ouverture (Jour 1), fin early bird et derniere heure avant fermeture. Concentre ton energie (et tes emails) sur ces 3 moments.
🚦
Gate Phase 5
Le lancement est termine
✓ Pre-launch execute ✓ Emails de lancement envoyes ✓ Contenu quotidien publie ✓ Early bird desactive ✓ Premieres ventes enregistrees

Post-Lancement & Optimisation

Mesurer, ameliorer, perenniser
Continu
▼
📊 Suivi des KPIs
1
Tracker les indicateurs cles
Inscriptions : nombre total + objectif
Taux de conversion : visiteurs page de vente → achat (objectif : 2-5%)
Taux de completion : % d'eleves qui finissent la formation (objectif : >60%)
CA genere : total + par canal (email, Instagram, organique)
Panier moyen : si plusieurs options de paiement
NPS / satisfaction : sondage post-formation
Kajabi Analytics Brevo Stats
⭐ Temoignages & preuve sociale
2
Collecter les temoignages
Envoyer un email a J+30 et a la fin du programme pour demander un temoignage.
Format ideal : video courte (30-60s) envoyee par WhatsApp.
Alternative : screenshot de message DM ou email (avec autorisation).
Template : "Qu'est-ce qui a change pour toi depuis que tu as suivi [formation] ?"
🔄 Optimisation
3
Analyser et ameliorer
Contenu : quels modules ont le plus de drop-off ? Re-filmer ou completer.
Page de vente : A/B test titre, CTA, prix si possible.
Emails : analyser les taux d'ouverture et de clic. Optimiser les objets.
Feedback : integrer les retours eleves dans la V2 du contenu.
♻️ Funnel evergreen
4
Mettre en place le funnel automatise
Si le lancement a fonctionne, convertir en funnel evergreen :
Lead magnet (PDF gratuit, mini-cours) → Sequence email (nurturing 5-7 jours) → Offre (avec deadline dynamique via Deadline Funnel ou similaire).
La formation se vend en continu sans avoir besoin de "lancer" a chaque fois.
Brevo Kajabi ManyChat
📅 Planification du prochain cycle
5
Planifier le prochain lancement
Cycles recommandes : 2 lancements par an (septembre + mars).
Reutiliser ce process pour chaque nouvelle formation.
Capitaliser sur les assets crees : emails, page de vente, automations — tout est reutilisable et ameliorable.
📝 Notes Catherine
Le vrai argent est dans l'optimisation, pas dans la creation de nouvelles formations. Regle : optimise une formation existante avant d'en creer une nouvelle. Un funnel evergreen bien huile peut generer des ventes chaque semaine sans effort supplementaire.
