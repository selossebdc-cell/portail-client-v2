Face Soul Yoga

Tes 3 premières actions sur Circle

Bienvenue dans la communauté Face Soul Yoga

Prends 5 à 10 minutes pour faire ces 3 actions simples. Elles te permettent de démarrer dans de bonnes conditions, à ton rythme.

1) Complète ton profil

Action : clique sur ton avatar puis sur ton profil.

  • Ajoute ton prénom.
  • Ajoute une photo (si tu veux).
  • Active les notifications utiles.

2) Va dans « Commencer ici »

Action : dans le menu de gauche, ouvre l’espace Commencer ici.

  • Lis le post de bienvenue épinglé.
  • Repère les 3 premières actions de la semaine.
  • Suis les étapes dans l’ordre, sans pression.

3) Publie ton premier message dans « Questions »

Action : ouvre l’espace Questions puis clique sur Nouveau post.

  • Présente-toi en quelques lignes.
  • Ou pose ta première question du moment.
  • Exemple : « Je débute, par quoi commencer cette semaine ? »
Besoin d’aide ?
Poste directement ta question dans l’espace Questions, on te répond rapidement.
